My Project
---
 
Welcome to a simple react-redux app with webpack, babel, npm, and express.
Feed your pet with fish, beef, chicken. Check the state using redux-devtools chrome.
 
 
Usage
---
 
Start the development server with this command:
 
```
npm start
```
 
 
 
Setup
---
 
```
npm install
```

 
 
 
Compile
---
 
```
npm run compile
```
